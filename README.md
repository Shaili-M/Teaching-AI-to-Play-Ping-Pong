__Installation__
1. Installing OpenAI Gym & install `cmake` first.
2. Install NumPy: `pip install numpy`
3. Install TensorFlow: `pip install tensorflow`
4. Install Keras: `pip install keras`

And now Run
$ python pingpong.py
